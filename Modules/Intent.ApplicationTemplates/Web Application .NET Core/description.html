A full technology stack for building a web server using ASP.NET Core in Visual Studio 2017+.

This template configures a hexagonal architecture with a technology agnostic Domain and Application layer.

The following technologies are used:

  • ASP.NET Core - framework for building HTTP services
  • Entity Framework Core - Microsoft's recommended ORM (Object Relational Mapper), providing an abstraction on the database.

Other notes:

  • Service Calls - Service calls are implemented using a lazy dispatch pattern which is abstracted away from the WebApi technology and technical concernts. Your service call business code should be implemented in the 'SCH' classes.
  • Repository Pattern - A collection oriented repository pattern provides access to the underlying database via Entity Framework. The DbContext backing each repository is isolated to each service call. The same instance is returned for any instantiation of a repository during the service call.
  • Dependency Injection - Dependency injection is implemented using the standard ASP.NET Core DI container.
